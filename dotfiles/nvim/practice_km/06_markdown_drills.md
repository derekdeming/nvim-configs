# Markdown Drills (markdown buffers only)

Open a markdown file for these mappings to work.

Part A: Toggle lists on current line (normal mode)
Step 1: Toggle number list
# Keys: t n
tn

Step 2: Toggle bullet list
# Keys: t b
tb

Step 3: Toggle checkbox
# Keys: t c
tc

Step 4: Toggle task state
# Keys: t t
tt

Step 5: Smart list toggle
# Keys: t l
tl

Part B: Toggle lists on selected lines (visual mode)
Step 6: Toggle number list on selection
# Keys: v (select lines), t n
v tn

Step 7: Toggle bullet list on selection
# Keys: v, t b
v tb

Step 8: Toggle checkbox on selection
# Keys: v, t c
v tc

Step 9: Toggle task state on selection
# Keys: v, t t
v tt

Step 10: Smart list toggle on selection
# Keys: v, t l
v tl

Part C: Task management
Step 11: Mark all tasks done
# Keys: Space t c
<leader>tc

Step 12: Mark all tasks undone
# Keys: Space t u
<leader>tu

Part D: Headings
Step 13: Toggle headings on current line
# Keys: Space h 1 (through 6)
<leader>h1
<leader>h2
<leader>h3
<leader>h4
<leader>h5
<leader>h6
