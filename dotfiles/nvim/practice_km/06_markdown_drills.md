# Markdown Drills (markdown buffers only)

Open a markdown file for these mappings to work.

Part A: Toggle lists on current line (normal mode)
Step 1: Toggle number list
# Keys: t n
tn
Press: t, then n.
Example: On a plain line, press t n to toggle a numbered list.

Step 2: Toggle bullet list
# Keys: t b
tb
Press: t, then b.
Example: On a plain line, press t b to toggle a bullet list.

Step 3: Toggle checkbox
# Keys: t c
tc
Press: t, then c.
Example: On a list item, press t c to toggle a checkbox.

Step 4: Toggle task state
# Keys: t t
tt
Press: t, then t.
Example: On a task item, press t t to toggle its state.

Step 5: Smart list toggle
# Keys: t l
tl
Press: t, then l.
Example: On a line, press t l to smart-toggle the list type.

Part B: Toggle lists on selected lines (visual mode)
Step 6: Toggle number list on selection
# Keys: v (select lines), t n
v tn
Press: v, select lines, then t, then n.
Example: Select multiple lines and press v t n to make a numbered list.

Step 7: Toggle bullet list on selection
# Keys: v, t b
v tb
Press: v, select lines, then t, then b.
Example: Select multiple lines and press v t b for a bullet list.

Step 8: Toggle checkbox on selection
# Keys: v, t c
v tc
Press: v, select lines, then t, then c.
Example: Select list items and press v t c to add checkboxes.

Step 9: Toggle task state on selection
# Keys: v, t t
v tt
Press: v, select lines, then t, then t.
Example: Select tasks and press v t t to toggle their state.

Step 10: Smart list toggle on selection
# Keys: v, t l
v tl
Press: v, select lines, then t, then l.
Example: Select lines and press v t l to smart-toggle lists.

Part C: Task management
Step 11: Mark all tasks done
# Keys: Space t c
<leader>tc
Press: Space, then t, then c.
Example: In a markdown file with tasks, press Space t c to mark all done.

Step 12: Mark all tasks undone
# Keys: Space t u
<leader>tu
Press: Space, then t, then u.
Example: Press Space t u to mark all tasks undone.

Part D: Headings
Step 13: Toggle headings on current line
# Keys: Space h 1 (through 6)
<leader>h1
Press: Space, then h, then 1.
<leader>h2
Press: Space, then h, then 2.
<leader>h3
Press: Space, then h, then 3.
<leader>h4
Press: Space, then h, then 4.
<leader>h5
Press: Space, then h, then 5.
<leader>h6
Press: Space, then h, then 6.
Example: Place cursor on a line and press Space h 2 to toggle an H2.
