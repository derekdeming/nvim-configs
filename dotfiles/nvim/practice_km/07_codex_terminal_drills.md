# Codex and Terminal Drills

Meta key reminder: <M-...> means Alt/Option + key.

Part A: Codex window controls (normal mode)
Step 1: Toggle Codex
# Keys: Space c c
<leader>cc
Press: Space, then c, then c.
Example: Press Space c c to toggle the Codex window.

Step 2: Focus toggle (Codex <-> host)
# Keys: Space c W
<leader>cW
Press: Space, then c, then Shift+w.
Example: Press Space c W to swap focus between Codex and the host window.

Step 3: Hide Codex
# Keys: Space c Q
<leader>cQ
Press: Space, then c, then Shift+q.
Example: Press Space c Q to hide Codex.

Step 4: Restart Codex
# Keys: Space c R
<leader>cR
Press: Space, then c, then Shift+r.
Example: Press Space c R to restart Codex.

Step 5: Resize Codex
# Keys: Space c - (narrow)
<leader>c-
Press: Space, then c, then - (dash).
# Keys: Space c + (widen)
<leader>c+
Press: Space, then c, then + (plus, Shift+=).
Example: Press Space c - to narrow and Space c + to widen.

Step 6: Open Codex in a specific directory
# Keys: Space c Shift+C
<leader>cC
Press: Space, then c, then Shift+c.
Example: Press Space c C and choose a directory.

Step 7: Update Codex CLI (brew)
# Keys: Space c U
<leader>cU
Press: Space, then c, then Shift+u.
Example: Press Space c U to update the Codex CLI.

Part B: Codex terminal mode (inside Codex)
Step 8: Window navigation
# Keys: Alt/Option + h/j/k/l
<M-h>
Press: Alt/Option+h.
<M-j>
Press: Alt/Option+j.
<M-k>
Press: Alt/Option+k.
<M-l>
Press: Alt/Option+l.
Example: In Codex terminal mode, use Alt/Option+h/j/k/l to move between splits.

Step 9: Window navigation with arrows
# Keys: Alt/Option + Arrow
<M-Left>
Press: Alt/Option+Left Arrow.
<M-Down>
Press: Alt/Option+Down Arrow.
<M-Up>
Press: Alt/Option+Up Arrow.
<M-Right>
Press: Alt/Option+Right Arrow.
Example: Use Alt/Option+Arrow keys to move between splits.

Step 10: Focus toggle, hide, restart (terminal mode)
# Keys: Alt/Option + w / q / r
<M-w>
Press: Alt/Option+w.
<M-q>
Press: Alt/Option+q.
<M-r>
Press: Alt/Option+r.
Example: In Codex terminal mode, press Alt/Option+w/q/r to control focus, hide, or restart.

Part C: Floating terminal (non-Codex)
Step 11: Toggle floating terminal
# Keys: Space c /
<space>c/
Press: Space, then c, then /.
Example: Press Space c / to open the floating terminal.

Step 12: Exit terminal mode
# Keys: Esc Esc
<Esc><Esc>
Press: Escape, then Escape.
Example: In terminal mode, press Esc Esc to return to normal mode.
