# Codex and Terminal Drills

Meta key reminder: <M-...> means Alt/Option + key.

Part A: Codex window controls (normal mode)
Step 1: Toggle Codex
# Keys: Space c c
<leader>cc

Step 2: Focus toggle (Codex <-> host)
# Keys: Space c W
<leader>cW

Step 3: Hide Codex
# Keys: Space c Q
<leader>cQ

Step 4: Restart Codex
# Keys: Space c R
<leader>cR

Step 5: Resize Codex
# Keys: Space c - (narrow)
<leader>c-
# Keys: Space c + (widen)
<leader>c+

Step 6: Open Codex in a specific directory
# Keys: Space c Shift+C
<leader>cC

Step 7: Update Codex CLI (brew)
# Keys: Space c U
<leader>cU

Part B: Codex terminal mode (inside Codex)
Step 8: Window navigation
# Keys: Alt/Option + h/j/k/l
<M-h>
<M-j>
<M-k>
<M-l>

Step 9: Window navigation with arrows
# Keys: Alt/Option + Arrow
<M-Left>
<M-Down>
<M-Up>
<M-Right>

Step 10: Focus toggle, hide, restart (terminal mode)
# Keys: Alt/Option + w / q / r
<M-w>
<M-q>
<M-r>

Part C: Floating terminal (non-Codex)
Step 11: Toggle floating terminal
# Keys: Space c /
<space>c/

Step 12: Exit terminal mode
# Keys: Esc Esc
<Esc><Esc>
